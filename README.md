# randomizer

### Prerequisites
1. Install java JDK 11
2. Install node and npm
3. Install Maven`

4. Clone pokered 
```
git clone https://github.com/pret/pokered.git ~/wd/pokemon/pokered
```

### Installation
Into ui direction to install web dependencies
```
make
```

### Execute CLI
```
make
```
    
### Execute website
```
cd ui
node app.js
```
